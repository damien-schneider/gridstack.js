Angular

Angular Wrapper

Gridstack.js is a modern pure Typescript library, with no external dependencies, that can work with any framework. It comes bundled with an Angular wrapper, and include many React|Vue examples (wrapper coming soon) as well as others.

We now ship an Angular Component to make it supper easy for that framework (Vue and React examples are above)

These are complete Angular projects with multiple options. use `yarn` and `yarn start` in angular demo sub-project to run them
